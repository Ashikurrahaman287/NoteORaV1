## Prompt: Build Noteora – Smart Data Analytics Web App

Build a professional, modern, and scalable web application called **Noteora – Smart Data Analytics Web App**.

**Noteora** is a smart data analytics and business intelligence platform that helps individuals, startups, and businesses track data, organize insights, visualize performance, and make better decisions through clean dashboards and structured reporting tools.

The main goal of this web app is to turn raw business data into clear, useful, and visual insights.

---

## Core Concept

Noteora should allow users to upload, manage, analyze, and visualize business data in a clean and user-friendly interface. The platform should help users understand trends, performance, growth, problems, and opportunities through dashboards, charts, reports, and smart summaries.

The app should be suitable for:

* Business owners
* Startup founders
* Marketing teams
* Sales teams
* Finance teams
* Operations teams
* Data analysts
* Small and medium businesses

---

## Main Features

### 1. User Authentication

Create a secure authentication system with:

* Email/password login
* Google login option
* Forgot password
* User profile management
* Role-based access system:

  * Admin
  * Manager
  * Analyst
  * Viewer

Each user should have their own private workspace.

### 2. Dashboard Overview

Create a powerful main dashboard where users can see their business performance at a glance.

Dashboard should include:

* Total records
* Revenue overview
* Expense overview
* Profit/loss summary
* Sales performance

* Monthly trends
* Recent activity
* Key performance indicators
* Top-performing categories
* Alerts for unusual data changes

Use clean cards, charts, progress indicators, and tables.

Users should be able to import data from different sources.

* Save datasets into projects

* Line chart

* Growth trend chart

Users should be able to:

* Export charts as image or PDF

Add an AI-powered insight section that explains the data in simple language.

The system should generate:

* Summary insights
* Trend analysis
* Warning alerts
* Growth opportunities
* Business recommendations

Create a professional report builder where users can generate business reports.
